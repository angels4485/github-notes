# github-notes
## What are Git and Github?
Git and Github are tools used by different developers in order to collaborate and create a project, innovation, etc.
Git is a version control that allows different people to work on the same file on real time without interrupting or ruining each other's work by accident. Github is a platfrom or webservice where you can run Git.

### Keyterms
Repository or "repo".
  * It's just another word for *project*; it can have multiple files associated with it.

Commit
  * It's like the word *save*
